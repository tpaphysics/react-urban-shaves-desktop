## **💻 Projeto**

O projeto <strong>Urban Shaves</strong> foi desenvolvido em [React](https://reactjs.org/) utilizando o [Chakra UI](https://chakra-ui.com/guides/first-steps).

```bash
# Instalação das dependências
$ yarn
  
# Start
$ yarn dev
```
